# CLI

| 명령어                         | 설명                                 |
| ------------------------------ | ------------------------------------ |
| `pwd`(print working directory) | 현재 위치                            |
| `cd`                           | 폴더 이동                            |
| `mkdir`                        | 폴더 생성                            |
| `touch `                       | 파일 생성                            |
| `rm `                          | 파일 삭제                            |
| `rm -r`                        | 폴더 삭제                            |
| `ls`                           | 현재 위치 폴더의 모든 하위 항목 표시 |
| ` ctrl + l`                    | 터미널 정리                          |
| `ctrl + c`                     | 취소                                 |
| `~ `                           | 홈 폴더 상징 기호                    |
| `/`                            | 최상단 폴더 상징 기호                |
| `.`                            | 현재 위치 상징 기호                  |
| `.. `                          | 현재 기준 상위 폴더 상징 기호        |
| `cat`                          | 파일 내용 출력                       |
| `mv` a b                       | 폴더/파일명 변경                     |
| `mv` [파일/폴더명] [경로]      | 파일/폴더 이동                       |
| `cp` [파일] [경로]             | 파일 복사                            |
